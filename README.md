# mon-premier-go

Petit projet d'exemple en Go.

Ce projet démarre un petit site web local avec le package `net/http`.

Exécution :

```powershell
cd C:\mon-premier-go
go run main.go
```

Puis ouvre un navigateur sur :

```text
http://localhost:8080
```

Tu peux ajouter d'autres pages en créant de nouveaux handlers ou en utilisant des templates.
